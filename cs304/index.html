CS304 - Databases with Web Interfaces

About

CS 304 is an introduction to web applications using relational databases and the SQL language, using MySQL middleware, including PHP, CGI/Python, and Flask front-end web pages using JavaScript, jQuery and Ajax.

Along the way, we discuss web application security, including SQL injection, XSS attacks, HTTPS and .htaccess.
